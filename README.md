# 🚀 Productivity Tracker Web App

A modern web-based productivity tracker built using Flask.
This app helps users manage daily tasks, track progress, and improve productivity.

---

## ✨ Features

* Add and manage tasks
* Morning planning & evening review
* Clean dashboard UI
* Task completion tracking

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS (Bootstrap)
* Jinja2 Templates

---

## 🚀 How to Run Locally

```bash
git clone https://github.com/damnnrachit/productivity-tracker.git
cd productivity-tracker
pip install -r requirements.txt
python app.py
```
